# Valeta Application Autofill companion

This unpacked Chrome extension fills common job-application fields from the JSON package exported by Valeta Job Seeker. It never clicks submit, never fills file uploads, and highlights sensitive fields for manual review.

## Install on a MacBook

1. Download or clone this repository.
2. Open `chrome://extensions` in Chrome.
3. Turn on **Developer mode**.
4. Choose **Load unpacked** and select this `extension` folder.
5. In Valeta Job Seeker, open **Autofill assistant** and download the JSON package.
6. Open the extension, paste the package, save it, then click **Fill this page** on an application form.

Always review every field before submitting. Application sites change frequently; unsupported fields remain untouched.
